# Learnify Frontend

### Setup instruction

1. Clone the project.
```
  git clone https://github.com/harshits1r8a/learnify-frontend.git
  
```

2. Move in to directory.
```
  cd learnify-frontend
  
```

3. Install dependencies.
```
  npm i
  
```

2. Run the server.
```
  npm run dev
  
```
